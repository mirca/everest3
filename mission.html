mission.py¶

Mission or telescope-specific routines. Each mission should be a subclass of the Mission object defined below.

class everest3.mission.Mission¶

A generic mission object.

download(target)¶

Downloads the raw data for a specific target.

Parameters:target (float, int, or str) – The target identifier
name¶

A string corresponding to the mission name.

path¶

The full path to the top-level mission data directory.

scatter(time, flux)¶

Computes the scatter metric for a light curve.

Parameters:
  • time (array_like) – The time array
  • flux (array_like) – The flux array
Return scatter:

The scatter metric for the light curve, in ppm
