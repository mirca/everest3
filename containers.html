class everest3.containers.RawTimeSeries(time, aperture, flux3D, errors3D=None)¶

A data container for a raw timeseries of fluxes and flux errors defined on a two-dimensional postage stamp.

Parameters:
  • time (array_like) – The array of times for each of the observations.
  • flux3D (array_like) – The raw 3D pixel postage stamps across the timeseries. This is an array with dimensions (ncols, nrows, ncads).
  • errors3D (array_like) – The raw 3D pixel postage stamp errors across the timeseries. This is an array with dimensions (ncols, nrows, ncads).
  • aperture (array_like) – A 2D binary array of shape (ncols, nrows) with 1’s corresponding to pixels included in the aperture and 0’s to pixels outside the aperture.
aperture¶

A 2D binary array of shape (ncols, nrows) with 1’s corresponding to pixels included in the aperture and 0’s to pixels outside the aperture.

errors¶

The SAP flux array, obtained by summing the errors in quadrature for each of the pixels in the aperture at each cadence.

errors2D¶

The flattened pixel errors timeseries _within the aperture_. This is a 2D array with dimensions (npix, ncads).

errors3D¶

The raw 3D pixel postage stamp errors across the timeseries. This is an array with dimensions (ncols, nrows, ncads).

flux¶

The SAP flux array, obtained by summing the fluxes in each of the pixels in the aperture at each cadence.

flux2D¶

The flattened pixel timeseries _within the aperture_. This is a 2D array with dimensions (npix, ncads).

flux3D¶

The raw 3D pixel postage stamps across the timeseries. This is an array with dimensions (ncols, nrows, ncads).

ncads¶

The number of cadences in the timeseries.

ncols¶

The number of columns in the postage stamp.

npix¶

The number of pixels within the aperture.

nrows¶

The number of rows in the postage stamp.

time¶

The time array.
