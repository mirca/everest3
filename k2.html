class everest3.k2.Target(*args, **kwargs)¶

Bases: everest3.containers.Target

A class that stores all the information, data, attributes, etc. for a K2 target de-trended with everest3.

ID¶

A str or int identifier for this target.

cadence¶

A float corresponding to the observation cadence in seconds.

detrend(*args, **kwargs)¶

De-trend the light curve via everest3.pld.detrend().

download()¶

Downloads the raw data for this target.

lightcurve¶

A Lightcurve object containing the dataset for this target.

mag¶

A float corresponding to the magnitude of this target in the mission band.

mission¶

The mission name.

scatter()¶

Computes the scatter metric for the light curve.

season¶

An int or float season/quarter/campaign identifier for this target.
